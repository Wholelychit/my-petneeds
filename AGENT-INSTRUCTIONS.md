# PetNeeds.ai Agent Instructions

## Stack
Use plain HTML, CSS, and JavaScript only.

Do not use React, Vite, TypeScript, npm, build tools, or unnecessary frameworks.

## Work Rules
1. Check existing files before editing.
2. Preserve the current working structure.
3. Do not rebuild from scratch unless a file is tiny or broken.
4. Prefer full-section or full-file updates for larger changes.
5. Avoid fragile tiny patches unless fixing a clear bug.
6. Keep navigation consistent across pages.
7. Keep footer links consistent across pages.
8. Add new public pages to sitemap.xml.
9. Update CHANGELOG.md after meaningful changes.
10. Never paste planning notes into user-facing code.
11. Never remove working features unless requested.
12. Keep copy simple and beginner-friendly.
13. Build mobile-first.
14. Make direct repo updates when access allows.

## Site Direction
PetNeeds.ai is a pet information and community website.

Priority areas:
- homepage
- main pet category pages
- emergency guide page
- Ask AI placeholder page
- upload pet photo page
- blog/article template
- privacy and disclosure pages

## Design Direction
Use a clean, warm, family-friendly style with readable text, large buttons, rounded cards, calm colors, and simple navigation.

## Owner Preference
Update files directly. Do not ask for repeated approval when the next step is clear.
